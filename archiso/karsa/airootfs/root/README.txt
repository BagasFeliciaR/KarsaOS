ðŸš€ HOW TO BUILD
----------------------------------------------
1. Make sure `archiso` is installed:
   # pacman -S archiso --needed

2. Clone this repository:
   $ git clone https://github.com/BagasFeliciaR/KarsaOS.git
   $ cd KarsaOS/archiso/karsa

3. Build the ISO:
   # mkarchiso -v -o ./out .

4. The generated ISO will be inside the `out/` folder.
